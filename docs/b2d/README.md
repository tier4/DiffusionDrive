These documents were created as part of the Bench2Drive dataset integration effort and contain information specific to working with the Bench2Drive dataset, HD maps, and BEV semantic generation features that are not part of the original DiffusionDrive codebase.

For general DiffusionDrive documentation, see the main `docs/` directory.
